# Iconography

### Example usage:

```tsx
<Monicon name='lucide:chevrons-up-down' />
```

#### Click on the icon to copy their code to clipboard.

<IconGallery>
  {Icons.map((icon) => (
    <IconItem name={icon}>
      <button className="hover:bg-slate-600" title={`Copy ${icon} to clipboard`} onClick={() => {
          navigator.clipboard.writeText(icon);
        }}
      >
        <Monicon name={icon} />
      </button>
    </IconItem>
  ))}
</IconGallery>
